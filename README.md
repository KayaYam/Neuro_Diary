# NeuroDiary

> Офлайн-первый дневник с LLM-анализом записей и 9 ролями-персонами.

NeuroDiary — это приложение для ведения личного дневника на Android. Записывайте мысли, используйте `#теги` для организации, получайте глубокий анализ и свежий взгляд на свои записи через встроенный ИИ-оракул.

## Быстрый старт

1. **Скачайте** [последнюю версию APK](https://github.com/KayaYam/Neuro_Diary/releases/latest)
2. **Установите** на устройство Android (8.0+)
3. **Зарегистрируйтесь** — email + пароль + код из письма
4. **Напишите первую запись** — нажмите **✎**
5. **Спросите Оракула** — выберите роль и нажмите кнопку

## Установка

### Системные требования
- Android 8.0 (Oreo, API 26) или выше
- ~150 MB свободного места

### Скачивание
1. Перейдите на [страницу релизов](https://github.com/KayaYam/Neuro_Diary/releases/latest)
2. Скачайте файл `NeuroDiary-vX.X.X.apk`
3. Откройте скачанный файл — система может запросить разрешение на установку из неизвестных источников

### Разрешения
- **Интернет** — для регистрации и запросов к Оракулу
- **Биометрия** — для блокировки экрана (опционально)

## Регистрация и вход

![Экран входа](screenshots/auth.png)

### Регистрация
1. На экране входа нажмите **«Регистрация»**
2. Введите **email** и **пароль** (минимум 6 символов)
3. Нажмите **«Зарегистрироваться»**
4. Проверьте почту — придёт письмо с **6-значным кодом**
5. Введите код и нажмите **«Подтвердить»**

Если письмо не пришло — проверьте папку «Спам» или подождите 1-2 минуты.

### Вход
1. Введите email и пароль
2. Нажмите **«Войти»**

### Восстановление пароля
1. На экране входа нажмите **«Забыли пароль?»**
2. Введите email, нажмите **«Отправить код»**
3. Проверьте почту — придёт код
4. Введите код и **новый пароль**
5. Нажмите **«Сбросить пароль»**

### Выход из аккаунта
Нажмите на email в правом верхнем углу (на главном экране) → **«Выйти»**.

## Главный экран — список заметок

![Главный экран](screenshots/main_screen.png)

На главном экране отображаются все ваши записи в виде карточек. Каждая карточка показывает дату и время, первые строки текста, подзаголовок (если есть) и #теги.

### Изменение порядка записей
Кнопка сортировки в верхней панели:
- **Сначала новые** (по умолчанию)
- **Сначала старые**

### Фильтр по тегу
Нажмите на `#тег` в любой карточке заметки — список покажет только заметки с этим тегом. Чтобы снять фильтр — нажмите на тег ещё раз или нажмите «×» рядом с названием тега.

### Быстрая запись
Нажмите **«+»** в правом нижнем углу — откроется редактор для новой заметки. Если сегодня уже есть запись, появится диалог: **«Сделать новую»** (отдельная запись) или **«Продолжить»** (дополнить существующую).

### FAB-меню
Кнопка **«+»** открывает меню: сменить роль, настроить свою роль, включить/выключить блокировку.

### Виджет
На домашнем экране телефона добавьте виджет NeuroDiary — он откроет редактор новой записи одним касанием.

### Системные уведомления
Красный колокольчик в верхней панели — есть непрочитанные системные сообщения. Нажмите, чтобы прочитать.

## Редактор заметок

![Редактор](screenshots/editor.png)

### Создание заметки
Нажмите **«+»** на главном экране — откроется пустой редактор.

**Подзаголовок** — верхнее поле (необязательно). Отображается в карточке на главном экране.

**Основной текст** — поддерживает обычный текст, `#теги` (выделяются цветом), эмодзи.

### Автодополнение тегов
Начните вводить `#` и первые буквы тега — появится выпадающий список с существующими тегами. Нажмите на нужный — тег вставится автоматически.

### Автосохранение
Заметка сохраняется автоматически через 5 секунд после последнего изменения. Полоска загрузки внизу показывает процесс сохранения.

### Маркер времени
Если вы продолжаете сегодняшнюю запись и прошло больше 30 минут, в текст добавится отметка:
```
--- в тот же день, в HH:mm
```

### Кнопка «Свернуть клавиатуру»
Внизу редактора — **▼ Свернуть клавиатуру**, чтобы скрыть клавиатуру и увидеть больше текста.

## Оракул — ИИ-анализ записей

![Оракул](screenshots/oracle.png)

Оракул отправляет вашу заметку нейросети и возвращает анализ, совет или свежий взгляд — в зависимости от выбранной роли.

### Как работает
1. Откройте заметку
2. Нажмите кнопку с названием роли (например, **«Оракул»**)
3. Появится диалог с таймером — **«Ожидание ответа... N сек»**
4. Можно свернуть экран или перейти в другие заметки — ответ придёт в фоне
5. Когда ответ готов, он откроется в диалоговом окне
6. **«Сохранить в заметку»** — ответ добавится в конец текущей заметки
7. Если закрыть диалог — ответ сохранится как отдельная заметка

### Что видит Оракул
При запросе Оракул получает текущую заметку, 5 последних записей, абзацы из других заметок по #тегам и книги из библиотеки по теме.

### Условия
- Кнопка видна только после входа в аккаунт
- Становится неактивной при балансе = 0
- Нужен интернет

## Своя роль

NeuroDiary позволяет создать собственную роль с уникальным характером.

### Настройка
1. Выберите **«Своя роль»** в списке ролей
2. Заполните поля:
   - **Книги** — через запятую книги, которые знает ваша роль
   - **Характер** — опишите персонажа («Ты строгий наставник», «Ты добрая бабушка»)
   - **Уровень поддержки** — ползунок от 1 до 10: 1–3 мягкая поддержка, 4–7 сбалансированный, 8–10 строгий/честный
   - **Домены** — через запятую сферы знаний (психология, философия, бизнес)
3. Нажмите **«Сохранить»**

После сохранения текущая роль автоматически станет «Своя роль». Настройки можно изменить через FAB-меню → **«Настроить свою роль»**.

## Теги

Просто пишите `#слово` в тексте заметки.

- **Длина** — до 50 символов
- **Состав** — буквы, цифры, дефис, подчёркивание
- **Завершение** — пробел или перевод строки
- **Регистронезависимые**: `#Работа` и `#работа` — один тег

При вводе `#` появляется автодополнение из существующих тегов. Нажмите на нужный — он подставится.

**Фильтрация**: нажмите на `#тег` в заметке на главном экране.

Неиспользуемые теги удаляются автоматически.

## Тёмная тема

Приложение автоматически подстраивается под системную тему. Отдельного переключателя нет.

## Блокировка экрана

![Блокировка](screenshots/lock.png)

### Как включить
1. На главном экране нажмите **«+»** → **«Включить блокировку»**
2. Подтвердите биометрию (отпечаток пальца или лицо)

### Как работает
Приложение блокируется при сворачивании. При возврате запрашивается биометрия. Можно использовать PIN-код или пароль устройства как запасной вариант.

### Как выключить
Разблокируйте приложение → **«+»** → **«Выключить блокировку»**.

**Требование**: на устройстве должен быть настроен хотя бы один отпечаток / лицо.

## Резервное копирование

### Экспорт
1. **«Поделиться»** на главном экране → **«Экспорт»**
2. Выберите место для JSON-файла (Google Диск, Яндекс.Диск, файловый менеджер)

### Импорт
1. **«Поделиться»** → **«Импорт»**
2. Выберите ранее сохранённый JSON-файл
3. Подтвердите: **очистить данные перед импортом?**
   - **Да** — текущие заметки удалятся
   - **Нет** — заметки добавятся к существующим (дубликаты по дате пропускаются)

### Поделиться заметкой
В редакторе нажмите **«Поделиться»** — отправьте текст в любое приложение.

Администратор может отправлять сообщения (техработы, новые функции, изменение тарифов). Новые сообщения отображаются красным колокольчиком в верхней панели. После прочтения колокольчик исчезает.

## Решение проблем

### Не приходит код подтверждения
Проверьте спам. Подождите 1-2 минуты. Попробуйте другой email. Если ничего не помогает — напишите в поддержку.

### Ошибка сети при входе
Проверьте интернет. Перезапустите приложение. Сервер может быть на обслуживании.

### Оракул не отвечает (долгий таймер)
Асинхронный запрос может занимать до 5 минут. Можно свернуть приложение — ответ придёт в фоне. Если не пришёл через 5 минут — отправьте запрос снова.

### Приложение вылетает
Проверьте Android 8.0+. Очистите кэш: Настройки → Приложения → NeuroDiary → Память → Очистить кэш. Если повторяется — напишите в поддержку.

### Сброс пароля не работает
Убедитесь, что вводите тот же email. Проверьте спам. Лимит — 3 запроса в час, попробуйте через 15 минут.

## Политика конфиденциальности

**Последнее обновление: июль 2026**

### Какие данные собираются

**На устройстве:**
- Тексты заметок, теги, настройки (тема, роль, блокировка) — хранятся локально в Room / DataStore
- Логи запросов к Оракулу (текст промпта, время) — хранятся локально

**На сервере:**
- Email, хеш пароля (bcrypt), Device ID, дата регистрации, баланс токенов
- Текст промптов — временно, удаляется после получения ответа

**Передача третьим лицам:**
- Текст заметки + контекст передаётся в Cloud.ru Foundation Models для генерации ответа
- Cloud.ru не использует данные для обучения моделей

### Что НЕ собирается
Геолокация, контакты, фото, медиафайлы, аналитика.

### Безопасность
Все соединения — HTTPS (порт 8443). Пароли — bcrypt. JWT — 183 дня. Блокировка экрана — биометрия Android.
