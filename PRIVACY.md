# Политика конфиденциальности NeuroDiary

**Последнее обновление: июль 2026**

## Какие данные собираются

### На устройстве
- **Тексты заметок** — хранятся локально в Room-базе данных на устройстве
- **Теги** — хранятся локально
- **Настройки** (тема, роль, блокировка) — хранятся в DataStore
- **Логи запросов** к Оракулу — хранятся локально (текст отправленного промпта, время)

### На сервере
- **Email** — для аутентификации и отправки кодов
- **Хеш пароля** (bcrypt) — для входа
- **Device ID** — для привязки устройства
- **Дата регистрации** — для администрирования
- **Баланс токенов** — количество оставшихся запросов
- **Текст промптов** (system_prompt + user_prompt) — временно хранятся в задачах до получения ответа, после удаляются

### Передача третьим лицам
- **Текст заметки + контекст** передаётся в Cloud.ru Foundation Models для генерации ответа Оракула
- Cloud.ru не использует переданные данные для обучения моделей

## Что НЕ собирается
- Геолокация
- Контакты
- Фото и медиафайлы
- Данные об использовании (аналитика)

## Безопасность
- Все соединения с сервером — HTTPS (порт 8443)
- Пароли хранятся в bcrypt-хеше (не в открытом виде)
- JWT-токен действителен 183 дня
- Блокировка экрана через биометрию Android
